# 📚 Study Hub

Study Hub is a responsive online learning platform built using React JS. It provides courses, workshops, blogs, and a simple login/register system for beginners.

## 🚀 Live Demo

https://priyadarshanikumari.github.io/studyhub/

## ✨ Features

- User Registration & Login
- Local Storage Authentication
- Responsive UI using Bootstrap
- React Router Navigation
- Workshop Page
- Blog Page
- Course Pages
  - React JS
  - JavaScript
  - Bootstrap
  - HTML & CSS
- Logout Functionality

## 🛠️ Tech Stack

- React JS
- JavaScript (ES6)
- Bootstrap 5
- React Router DOM
- React Icons
- Vite

```
src

```

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Priyadarshanikumari/studyhub.git
```

Go to project folder

```bash
cd studyhub
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
